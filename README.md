To get this up and running:


`Cd into project folder` and run the commands below:

`pipenv shell`

## API Routes
`/api/forms`
GET|POST

`api/forms/<id>`
GET|UPDATE|DELETE

`api/forms/<id>/data`
GET|POST

`api/forms/<id>/data/<id>`
GET|UPDATE|DELETE
